#  Spotify Clone

A full-featured **Spotify Clone** built using **HTML, CSS, JavaScript, and Node.js (with Express.js)**.  
This project mimics the look and basic functionality of Spotify, allowing users to browse, play, and control songs seamlessly through a clean, responsive UI.

# Features

-  Play, pause, next, and previous track controls  
-  Organized playlist system  
-  Folder-based song loading using backend routes  
-  Dynamic audio player with real-time progress bar  
-  Responsive design ‚Äî works perfectly across devices  
-  Fast and smooth UI animations
